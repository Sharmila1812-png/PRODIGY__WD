 Stopwatch Web Application

A simple, interactive, and user-friendly stopwatch application built with HTML, CSS, and JavaScript. This stopwatch application allows users to start, pause, reset, and track lap times with precision. Designed for usability, it features a clean and intuitive interface that enables easy time tracking for various tasks.

## Features

- **Start/Stop Functionality**: Begin and pause the stopwatch with a single click.
- **Reset Option**: Clear the current time and reset the stopwatch to zero.
- **Lap Time Tracking**: Capture lap times to record and view intervals without interrupting the main timer.
- **User-Friendly Interface**: Simple and intuitive design for a seamless user experience.

## Technologies Used

- **HTML**: Used for structuring the application layout and elements.
- **CSS**: Provides styling for the application to enhance visual appeal and improve user interaction.
- **JavaScript**: Powers the functionality, including time calculations and controls for starting, pausing, resetting, and recording lap times.


### Usage

- **Start**: Click the "Start" button to begin timing.
- **Pause**: Click the "Pause" button to pause the timer without resetting.
- **Lap**: Click the "Lap" button to record and display a lap time below the timer.
- **Reset**: Click the "Reset" button to clear all laps and reset the timer to zero.
